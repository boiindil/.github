# Sponsorship and Funding

This repository is part of an independent open reference line by boiindil / ruahAI.

Sponsorships support the preparation of small, reviewable public artifacts around audit-ready governance, decision logic, computational traces, and reproducible prototypes.

Sponsorship does not grant commercial usage rights, private IP access, exclusivity, support obligations, or operational deployment rights.

For review, licensing, internal-use, OEM, embedded, white-label, or exclusive options, a separate written agreement is required.
